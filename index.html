Enhancing Usability and User Experience:

The Solar Design Optimizer (SDO) Redesign

A comprehensive UI/UX redesign of a critical desktop application for solar design engineers

Role

Research, UX & UI Designer

Timeframe

February 2024

Tools

Figma, Balsamiq

Background

Sungevity, a leading residential solar company, relied heavily on the Solar Design Optimizer (SDO) — a desktop application pivotal for its Solar Design and Engineering team. The team used it to design solar systems and generate detailed plan sets for permit approval.

The SDO was a feature robust app, but users reported significant usability issues: the interface was cluttered, visually tiring, and tough to navigate. Despite plans for a redesign to tackle these issues, Sungevity shut down operations in 2020 before changes could be made.

Personal Initiative

Despite Sungevity ceasing operations, I decided to independently redesign the SDO interface to showcase my design problem solving skills and illustrate how user-centric solutions could have revolutionized the experience. User feedback gathered prior to the closure was used to inform the redesign.

The Challenge

The SDO suffered from a lack of visual hierarchy and differentiation, featuring a monochromatic color palette and uniform typography that increased cognitive load and reduced user engagement in a task-intensive environment.

Additionally, the mismatch with Sungevity's vibrant brand identity was a missed opportunity to enhance user experience and reflect the company's motto: "Generate Positive".

Business Opportunity

The SDO is pivotal in streamlining the solar system design and permit approval process for residential solar installations, enhancing both profitability and operational efficiency. Here's how the redesign enhances key aspects:

  • Boost Permit Approval Percentages: Enhancing interface clarity helps minimize errors that impact permit approvals, expediting project timelines and reducing costly permit rejections.
  • Boosts Monthly Plan Set Completion Rates: Enhanced UI features allow designers to complete plan sets more efficiently, which directly boosts productivity and impacts revenue.

These improvements in usability and efficiency ensure the SDO redesign elevates key performance metrics and significantly enhances revenue generation.

Research & Understanding

User Feedback

At Sungevity, I regularly engaged with users to gather insights and identify pain points through various communication channels such as chat, email, and in-person discussions. The collected feedback was documented and systematically analyzed.

Previous feedback from saved documentation was gathered to use as qualitative data for review of user experiences and pain points.

User feedback was organized using affinity mapping to identify common themes and prioritize key areas for improvement.

Visual Usability Audit

I conducted a comprehensive Visual Usability Audit, using a design walkthrough checklist to evaluate aspects such as color palette, typography, iconography, spacing and layout, visual hierarchy, accessibility, and interaction cues. Questions included:

  • "Do the colors enhance readability?"
  • "Are the text sizes and styles used effectively and consistently?"
  • "Is the most important information or action visually prominent? Does it guide the user naturally through the tasks?"
  • "Does the interface meet WCAG accessibility standards?"
  • "Does the interface provide visual feedback in response to user actions?"

Analysis

User feedback and insights from the Visual Usability Audit identified several critical areas for improvement in the SDO, affecting usability and user satisfaction:

  • Layout and Navigation: The cluttered and disorganized layout, coupled with confusing navigation, made it challenging for users to efficiently find and interact with necessary tools.
  • Readability Issues: Users consistently reported difficulties with readability and eye strain, highlighting deficiencies in the color palette and typography.
  • Inconsistent User Experience: A lack of a cohesive design system has led to a fragmented and inconsistent interface across the app.

Addressing these key issues by refining the color palette, typography, layout, navigation, and implementing a comprehensive design system emerged as clear target areas needed to enhance usability and user satisfaction.

Design & Development

North Star Goal

Create an intuitively navigable, visually engaging software interface that minimizes cognitive load and maximizes efficiency, thereby transforming the complex process of solar system design into a streamlined, positive user experience that reflects the company mission to 'Generate Positive' in every interaction.

Style Guide

After pinpointing color and typography as key factors in boosting readability and usability, I dove into refining these elements.

I tapped into Sungevity's existing style guide. This strategy had dual benefits:

  • Brand Alignment: Ensuring the SDO user experience matches the company's upbeat vibe.
  • Enhanced readability: Sungevity's color palette is ideal for enhanced readability, as well as meeting WCAG color contrast accessibility standards.

Component Iterations

Input Fields

The focus shifted to the input fields, a crucial component given the SDO's heavy reliance on user data entry. I designed and continuously refined these fields right on the Structural screen, always testing and tweaking based on how they performed on the full screen layout.

Design Goal: Enhance readability

Intended Outcomes:

  • Lower Cognitive Load: Boost user efficiency and satisfaction.
  • Reduce User Errors: Minimize risk of users entering data into the wrong fields.

Buttons

Previously, all buttons in the SDO were uniform in size, shape, color, and label text, which led to user errors such as accidentally clicking "Delete" instead of "Edit".

Additionally, users often overlooked valuable workflow aid features due to indistinguishable button types.

Design Goal: Visual Distinction

Introducing five visually distinct categories of buttons—Action, Feature Launch, Task Modal, Global Action, and Icon Buttons—each uniquely designed to clarify its function, eliminated confusion and errors.

Final Screen Designs

Structural Screen

  1. Sidebar Navigation: Transformed an unused column into a functional sidebar navigation with dynamic sub-menus for quick access to main sections, optimizing space.
  2. Color Scheme, Typography, and Button Style Update: Adopted Sungevity's brand color palette, introduced systematic typography, and button design to improve visual clarity and hierarchy.
  3. Enhanced Tab Management: Added delete and add buttons to tabs with clear interactive states, improving user control and minimizing accidental deletions.
  1. Unified Container Layout: Simplified the screen structure by consolidating multiple nested containers into a single, two-section container for Roof Plane and Array.
  2. Optimized Input Field Layout: Redesigned the input fields to include more space between elements and aligned columns, reducing visual clutter.
  3. Relocation of Key Features: Shifted the 'Solar Access Measurements' table to the main screen and the 'Build Mount Detail' button to the Racking Calculator modal for better workflow alignment.

Electrical Screen

  1. Vertical Layout Reorganization: Transitioned from a horizontal to a vertical layout, aligning user workflow from top to bottom for more intuitive navigation.
  2. Consolidation of Configuration Controls: Positioned the "Pre-Configure" button alongside related dropdowns and checkboxes for logical grouping of controls.
  3. Integrated Calculation Tools: Placed calculators directly beneath configuration options for immediate access during setup.
  1. In-line Editing in Equipment Table: Added direct edit/delete controls within each equipment row to streamline the editing process.
  2. Drag Handles for Easy Reordering: Implemented drag handles for simple rearrangement of equipment items, enhancing workflow flexibility.
  3. Redesigned Action Buttons: Updated "Add Equipment" and "Draw ELD" buttons with distinct styles to clearly differentiate functions.
